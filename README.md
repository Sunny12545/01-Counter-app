MERN COUNTER APP
-A simple full-stack counter application built using the mern stack the application stores the counter value in mongodb and updates through react js.

Features:
-Display current counter value
-Increment counter value
-Counter value stored permanently in mongodb
-Rest Api using express
-React frontend connected to backend using Axios

Tech Stack:
-Frontend
React
Axios
Css
-Backend
Nodejs
Expressjs
MongoDB
Mongoose
Cors
Dotenv

Installation:
-Frontend
cd Frontend
npm i axios
node server.js
-Backend
cd Backend
npm install express mongoose dotenv cors
npm run dev
